# ICNaaS
Information-Centric Networking as a Service

## mcn-icn-sm
Contains the source code for the Service Manager handling the ICN service.

## mcn-icn-so
Contains the source code for the Service Orchestrator handling ICN service instances.

## mcn-icn-manager
Contains the source code for the ICN Manager component, responsible for handling registration, de-registration and topology coordination of CCN routers.

## mcn-ccn-router
Contains the original project CCNx's source code modified to run in the cloud and integrate with other services and functionalities, together with a small component (ccnserver) to interact with Follow-Me Cloud (FMC) functionalities.
